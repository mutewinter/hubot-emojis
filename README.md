# hubot-emojis

A hubot script for finding emoji.

See [`src/emojis.coffee`](src/emojis.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-emojis --save`

Then add **hubot-emojis** to your `external-scripts.json`:

```json
["hubot-emojis"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
